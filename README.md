
# ReliefZone

ReliefZone is a mental wellness platform designed to help users achieve emotional clarity through mood tracking, mindfulness tools, and guided clinical support.

## Key Features
- **Daily Mood Check-ins**: Track your emotional journey with AI-generated guidance.
- **Mindfulness Tools**: Breathing exercises, journaling, and thought reframing.
- **Clinical Support**: Connect with doctors for personalized wellness notes.
- **Multilingual**: Support for English, Afan Oromo, and Amharic.
